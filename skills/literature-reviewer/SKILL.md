# Literature Reviewer — 文献检索与综述撰写

详见本地 `~/.workbuddy/skills/literature-reviewer/SKILL.md`或 GitHub 仓库。

本 Skill 负责文献检索策略生成、文献综述撰写和研究空白识别。不执行实际的网络文献检索，而是对用户提供的文献材料进行分析和综合。
